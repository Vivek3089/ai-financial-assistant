#  AI Financial Assistant

An AI-powered financial assistant that provides investment insights using Large Language Models (LLMs) and real-time stock data.

##  Features
-  Real-time stock analysis (AAPL, TSLA, etc.)
-  AI-generated financial advice
-  Risk assessment
-  Beginner investment guidance
-  Fast responses using Groq API

##  Tech Stack
- Python
- Groq API (LLM)
- Gradio (UI)
- yFinance (Stock Data)

##  How it Works
User input → LLM processing → Real-time stock data → Structured financial response

##  Demo Questions
- What is a mutual fund?
- Analyze AAPL stock
- What are risks in investing?

##  Note
This project uses a pre-trained LLM via API and does not involve training a custom model.
