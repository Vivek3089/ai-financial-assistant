# AI Financial Assistant

An AI-powered financial assistant that provides intelligent investment insights using Large Language Models (LLMs) and real-time stock data.

---

## Features

- Real-time stock analysis (AAPL, TSLA, MSFT, etc.)
- AI-generated financial advice using LLMs
- Risk assessment for investments
- Beginner-friendly investment guidance
- Fast responses using Groq API
- Interactive UI with quick action buttons

---

## Tech Stack

- Python
- Groq API (LLM)
- Gradio (User Interface)
- yFinance (Real-time stock data)

---

## How It Works

User Input → LLM Processing → Real-Time Stock Data → Structured Financial Response

---

## Key Concepts

- Prompt Engineering  
- LLM Integration via API  
- Real-time Data Processing  
- Interactive UI Design with Gradio  

---

## How to Run

1. Install dependencies:
pip install groq gradio yfinance

2. Set your API key:
export GROQ_API_KEY=your_api_key_here

3. Run the project:
- Open the notebook in Google Colab  
- Run all cells  
- Launch the Gradio interface

## Example Questions

- What is a mutual fund?  
- Analyze AAPL stock
